Démonstration 7 La gestion des erreurs dans les JSP

Le lien suivant pointe vers la JSP jspGenerantErreur.jsp. Elle est configuré pour déclencher l'appel de la JSP jspTraitantErreur.jsp. /modules/module3/demonstrations/jspGenerantErreur.jsp

Voici le code de la JSP jspGenerantErreur.jsp : 

	 	
<%@ page language="java" 
	contentType="text/html; charset=UTF-8"
    pageEncoding="UTF-8"
	errorPage="jspTraitantErreur.jsp"
%>
<html>
<head>
<meta http-equiv="Content-Type" content="text/html; charset=UTF-8">
<title>Une page avec une erreur qui ne s'affichera jamais</title>
</head>
<body>
	<%
		String uneVariable=null;
		uneVariable.length();//NullPointerException
	%>
</body>
</html>

Voici le code de la JSP jspTraitantErreur.jsp : 

	 	
<%@ page language="java" 
	contentType="text/html; charset=UTF-8"
    pageEncoding="UTF-8"
    isErrorPage="true"%>
<html>
<head>
<meta http-equiv="Content-Type" content="text/html; charset=UTF-8">
<title>Une page gérant les erreurs survenues dans les JSP</title>
</head>
<body>
	<h1>Erreur :</h1>
	<p>
		(<%=exception %>)<%=exception.getMessage() %>
	</p>
	<a href="<%=request.getContextPath()%>/modules/module3/demonstrations/demonstration7.html">Retour</a>
</body>
</html>
