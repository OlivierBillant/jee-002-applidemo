TP 1 Chifoumi (version 3)

Enoncé

Apporter une évolution à l'application Chifoumi réalisée dans le module 3.

Description fonctionnelle

L'application peut s'adapter à la langue de l'utilisateur. Par défaut, la langue est celle configurée dans le navigateur. Si l'utilisateur choisit au travers d'un menu une autre langue, alors c'est celle-ci qui est préférée.

Eléments d'architecture

Ne pas hésiter à faire évoluer l'application pour reprendre les outils vus en cours (utilisation de JSP avec l'EL et les balises JSTL).
