Démonstration 1 La traduction des messages

Afficher des messages en lien avec la langue choisie

Le lien suivant pointe vers une JSP affichant le contenu avec la langue du navigateur ou la langue par défaut si les langues du navigateur ne sont pas disponibles: /modules/module8/internationalisationMessages?nom=Archibald.

Le lien suivant pointe vers une JSP affichant le contenu avec la langue anglaise : /modules/module8/internationalisationMessages?langue=en&nom=Charles.

Le lien suivant pointe vers une JSP affichant le contenu avec la langue par défaut (car le chinois n'existe pas dans l'application): /modules/module8/internationalisationMessages?langue=zh&nom=Kim.

Source des différents fichiers

  1. mes_messages.properties
    2. 				 	
msg_bonjour=Bonjour {0}!
msg_bonheur=Je suis heureux!
  2. mes_messages_en.properties
    3. 				 	
msg_bonjour=Hello {0}!
msg_bonheur=I am happy
  3. internationalisationMessages.jsp présent dans le répertoire /WEB-INF/modules/module8
    4. 				 	
<%@page import="java.util.Locale"%>
<%@ page language="java" contentType="text/html; charset=UTF-8"
    pageEncoding="UTF-8"%>
<%@ taglib prefix="c" uri="http://java.sun.com/jsp/jstl/core" %>
<%@ taglib prefix="fmt" uri="http://java.sun.com/jsp/jstl/fmt" %>
<!DOCTYPE html PUBLIC "-//W3C//DTD HTML 4.01 Transitional//EN" "http://www.w3.org/TR/html4/loose.dtd">
<html>
<head>
<meta http-equiv="Content-Type" content="text/html; charset=UTF-8">
<title>Internationalisation des messages</title>
</head>
<body>
	<c:if test="${!empty param.langue}">
		<!-- Définition de la Locale à utiliser -->
		<fmt:setLocale value="${param.langue}"/>
	</c:if>
	
	<p>
		<!-- Définition des fichiers ressources à utiliser -->
		<fmt:bundle basename="fr.eni.javaee.module8.mes_messages">
			<fmt:message key="msg_bonheur"></fmt:message>
		</fmt:bundle>
	</p>
	
	<p>
		<fmt:setBundle basename="fr.eni.javaee.module8.mes_messages" var="r"/>
		<!-- Lecture d'un message -->
		<fmt:message key="msg_bonjour" bundle="${r}">
			<fmt:param value="${param.nom}"></fmt:param>
		</fmt:message>
	</p>
			
	<p>
		<a href="demonstrations/demonstration1.html">Retour</a>
	</p>
</body>
</html>
  4. web.xml pour la configuration de l'accès à la JSP
    5. 				 	
<servlet>
  	<servlet-name>JSPInternationalisationMessages</servlet-name>
  	<jsp-file>/WEB-INF/modules/module8/internationalisationMessages.jsp</jsp-file>
</servlet>
<servlet-mapping>
	<servlet-name>JSPInternationalisationMessages</servlet-name>
	<url-pattern>/modules/module8/internationalisationMessages</url-pattern>
</servlet-mapping>
