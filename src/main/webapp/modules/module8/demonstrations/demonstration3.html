Démonstration 3 Le formatage des nombres

Afficher un nombre décimal avec 2 chiffres exactement après la vigule

Le lien suivant pointe vers une JSP affichant le résultat de la division 125/35 : /modules/module8/formatageNombres.

  1. formatageNombres.jsp présent dans le répertoire /WEB-INF/modules/module8
    2. 				 	
<%@page import="java.util.GregorianCalendar"%>
<%@ page language="java" contentType="text/html; charset=UTF-8"
    pageEncoding="UTF-8"%>
<%@ taglib prefix="c" uri="http://java.sun.com/jsp/jstl/core" %>
<%@ taglib prefix="fmt" uri="http://java.sun.com/jsp/jstl/fmt" %>
<!DOCTYPE html PUBLIC "-//W3C//DTD HTML 4.01 Transitional//EN" "http://www.w3.org/TR/html4/loose.dtd">
<html>
<head>
<meta http-equiv="Content-Type" content="text/html; charset=UTF-8">
<title>Formatage des nombres</title>
</head>
<body>
	<h2>Résultat de la division 125/35</h2>
	<p>
		Avec 2 chiffres exactement après la virgule :
		<fmt:formatNumber
			type="number"
			value="${125/35}"
			maxFractionDigits="2"
			minFractionDigits="2">
		</fmt:formatNumber>
	</p>
</body>
</html>
  2. web.xml pour la configuration de l'accès à la JSP
    3. 				 	
<servlet>
    <servlet-name>JSPformatageNombres</servlet-name>
    <jsp-file>/WEB-INF/modules/module8/formatageNombres.jsp</jsp-file>
</servlet>
<servlet-mapping>
  	<servlet-name>JSPformatageNombres</servlet-name>
  	<url-pattern>/modules/module8/formatageNombres</url-pattern>
</servlet-mapping>
