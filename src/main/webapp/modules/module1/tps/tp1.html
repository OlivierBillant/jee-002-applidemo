TP 1 Mise en place d'un template

Enoncé

A partir d'un nouveau Dynamic Web Project, proposer une interface de l'application suivante.

Description fonctionnelle

L'application doit permettre, au travers de liens hypertextes, d'accéder aux différentes applications que vous allez réaliser dans le cadre du cours sur le développement web côté serveur avec Java EE.

Cette application n'a pas vocation à être dynamique ou paramétrable.

Vous pourrez cependant la faire évoluer au fur et à mesure du cours avec les technologies abordées.
