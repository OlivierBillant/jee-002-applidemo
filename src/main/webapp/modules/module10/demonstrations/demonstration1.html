Démonstration 1 Le déploiement implicite

Génération du WAR

Faites-un clic droit sur le projet à déployer est cliquez sur le menu Export => WAR file. L'écran suivant apparait pour définir les caractéristiques de l'export:

Déploiement de l'application sur un Tomcat autonome

Commencez par éteindre le serveur Tomcat piloté par votre IDE.

Déposez ensuite votre archive dans le répertoire webapps de votre Tomcat:

Démarrez ensuite Tomcat à l'aide du script startup.bat présent dans le répertoire bin de Tomcat. Observez le déploiement de l'archive :

Naviguez vers votre application avec l'URL suivante : http://localhost:8080/AppliDemo

Si vous souhaitez que votre application soit accessible au travers de l'URL http://localhost:8080/TutoJavaEE, il suffit de renommer l'archive TutoJavaEE.war et de redémarrer Tomcat.
